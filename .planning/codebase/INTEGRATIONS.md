## External Data Sources

**AIS Vessel Movement Data:**
- Source: `given_data/vessel_movements_dataset.csv`
- Loaded by: `src/data_loader.py` via `load_vessel_movements_ships()`
- Contains: vessel ID, position, speed, timestamps, port boundaries, anchorage status

**Calculation Factors:**
- Source: `given_data/calculation_factors.xlsx`
- Contains: emission factors, LCV values, fuel prices, CAPEX calculations

**LLAF Table:**
- Source: `given_data/llaf_table.csv`
- Contains: Load vs. Atmospheric Emission Factors lookup

**Hackathon Submission Format:**
- Template: `given_data/submission_template.csv`
- 11 required columns for official submission
- Generated by: `src/optimization.py` via `submission_outputs()`
